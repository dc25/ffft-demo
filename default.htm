Multiplication is Easier When It's Complex

The purpose of this page is to demonstrate the Schönhage-Strassen technique, the use of Fast Finite Fourier Transforms to perform multiplication in n(log(n)) time. Enter two integers in the input windows. The product of the two is updated in real time along with all intermediate calculations. For an explanation of this technique and all the notation used on this page see this article. Email me if you have any comments or questions.
